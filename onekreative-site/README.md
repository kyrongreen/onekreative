# One Kreative — Landing Site

Single-file landing page for **One Kreative**, an Austin-area creative agency running a CO-OP EDDM mailer program.

## What This Is

A static, single-page site (`index.html`) that explains the EDDM mailer program, shows pricing for the Business and Community layouts, and includes CTAs to reserve an ad spot.

- 9×12 postcard mailed via USPS to ~2,500 homes every two weeks
- One business per category per mailer (exclusivity)
- One Kreative handles design; businesses pay per drop

## Design

Warm editorial — cream/gold palette, Playfair Display + Inter. No build step, no dependencies.

## Deploy

Drop this repo into Netlify (or any static host). Publish directory: `/` (root). No build command needed.
